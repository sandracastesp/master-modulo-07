# 🤖 Clon ChatGPT 

Este repositorio contiene un proyecto desarrollado en **React**, dividido en diferentes entregas, cada una con objetivos específicos que abarcan el uso de librerías, hooks y la integración de un servidor.

---

## ✨ Parte 1: Configuración inicial  
- Creación del repositorio en **GitHub**.  
- Organización del trabajo mediante **ramas** (`parte-1`).  
- Configuración del entorno local.  
- Integración de **Tailwind CSS** para estilos.  
- Implementación de **React Hook Form** para validación de formularios.  

---

## ✨ Parte 2: Clon básico de ChatGPT  
- Creación de un clon básico de **ChatGPT**.  
- Instalación y configuración de **Ollama**.  
- Uso de un **Custom Hook** en React para consumir el servicio de Ollama.  
- Descarga e integración de **DevSeek**.  
- Creación de componentes para gestionar el **historial de interacciones** (`History`).  
- Implementación de **Context API** y **useReducer** para manejar el estado global.  

---

## ✨ Parte 3: Context API en profundidad  
- Comprensión de la importancia de **useContext** (evitar *prop drilling*).  
- Creación de un **contexto** con `createContext`.  
- Implementación de un **Provider** para envolver la aplicación.  
- Uso de `useContext` en los componentes que necesitan acceder al estado.  

---

## ✨ Parte 4: Introducción a Express  
- Creación de un proyecto **Express** en una carpeta independiente.  
- Inicialización del proyecto con **NPM**.  
- Implementación de un servidor básico con un endpoint que responde **"Hola Mundo"**.  
- Pruebas para verificar el correcto funcionamiento del servidor.  

---

## 🚀 Tecnologías utilizadas  
- **React**  
- **Tailwind CSS**  
- **React Hook Form**  
- **Context API & useReducer**  
- **Ollama + DevSeek**  
- **Express**

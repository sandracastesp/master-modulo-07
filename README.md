Proyecto-uno: Esta es la primera entrega del proyecto de FrontEnd Avanzado con React. El objetivo principal de esta entrega fue crear un repositorio y aprender a trabajar de forma organizada usando ramas, se configuró el entorno local (repositorio en Github) y se trabajó en una rama específica para esta entrega  (`parte-1`). En dicha rama se integró Tailwind CSS para los estilos y React Hook Form para la validación de formularios.
